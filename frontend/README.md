# React + TypeScript + Vite

Este projeto usa Vite para desenvolvimento rapido com React e TypeScript.

## Como rodar
```bash
cd frontend
npm install
npm run dev
```

## Plugins oficiais
- `@vitejs/plugin-react`: usa Babel para Fast Refresh
- `@vitejs/plugin-react-swc`: usa SWC para Fast Refresh

## Observacoes
- A configuracao do ESLint pode ser expandida conforme a necessidade.
- Para detalhes de arquitetura DDD, veja `frontend/docs/README.md`.
 - Toda a documentacao do frontend esta em `frontend/docs/`.
