# Design e Identidade Visual

## Objetivo
Criar uma interface acolhedora, simples e colorida, com base nas cores associadas ao autismo.

## Tipografia
- **Nunito**: textos gerais
- **Baloo 2**: titulos e destaques

## Paleta
- Azul: `#1e88e5`
- Amarelo: `#fdd835`
- Verde: `#43a047`
- Vermelho: `#e53935`
- Texto: `#1e1f29`
- Fundo: `#f6f4ff`

## Logo e fundo
- Logo: `src/assets/logo-mundinho.png`
- Fundo: `src/assets/bg-mundinho.png`

O fundo usa a imagem + um gradiente suave para manter legibilidade.

## Estilos principais
- `src/index.css`: estilos globais, fundo e fontes
- `src/App.css`: layout da pagina, cards, formularios

## Formularios
- Inputs e selects ocupam 100% da largura do card
- Campos organizados em grid responsivo
