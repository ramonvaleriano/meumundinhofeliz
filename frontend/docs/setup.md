# Setup do Frontend

## Requisitos
- Node 18+
- npm

## Instalar dependencias
```bash
cd frontend
npm install
```

## Rodar em desenvolvimento
```bash
npm run dev
```

## Rotas disponiveis
- `/` (login)
- `/criar-conta` (cadastro)

## Build
```bash
npm run build
```

## Preview
```bash
npm run preview
```
