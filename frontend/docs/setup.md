# Setup do Frontend

## Requisitos
- Node 18+ (ou superior)
- npm

## Instalar dependencias
```bash
cd frontend
npm install
```

## Rodar em desenvolvimento
```bash
npm run dev
```

Abra no navegador:
- http://localhost:5173

## Build de producao
```bash
npm run build
```

## Preview do build
```bash
npm run preview
```

## Rotas disponiveis
- `/` (login)
- `/criar-conta` (cadastro)

## Dica para iniciantes
Sempre que mudar o `.env`, pare o servidor e rode `npm run dev` novamente.
