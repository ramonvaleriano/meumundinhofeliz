# Glossario (Leigo) - Frontend

- **Frontend**: a parte do sistema que aparece na tela.
- **Pagina**: cada tela que o usuario acessa (ex: login).
- **Formulario**: conjunto de campos para preencher.
- **Rota**: endereco interno do site (ex: `/criar-conta`).
- **Componente**: um pedaco reutilizavel de tela (botao, card, input).
- **React**: biblioteca usada para construir as telas.
- **Vite**: ferramenta que roda o projeto localmente.
- **TypeScript**: JavaScript com tipos, reduz erros.
- **.env**: arquivo onde ficam configuracoes, como URL da API.
