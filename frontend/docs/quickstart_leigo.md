# Guia Rapido (Leigo) - Frontend

## 1) Instalar e rodar
```bash
cd frontend
npm install
npm run dev
```

## 2) Abrir no navegador
- http://localhost:5173

## 3) O que voce vai ver
- Tela de login
- Botao “Criar conta”

## 4) Criar conta
- Clica em “Criar conta”
- Vai abrir um formulario com dados pessoais e endereco

## 5) Se der erro
- Verifique se o backend esta ligado (porta 8000)
- Veja mensagens no terminal onde rodou `npm run dev`
