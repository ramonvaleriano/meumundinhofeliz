# Visao Geral do Banco

Este documento apresenta todas as tabelas criadas no banco e aponta para suas documentacoes detalhadas.

## Tabelas
- `address` -> docs/address.md
- `profile_type` -> docs/profile_type.md
- `feature_flag` -> docs/feature_flag.md
- `users` -> docs/users.md

## Seeds automaticos
- `profile_type`: preenche lista de profissionais
- `feature_flag`: preenche flags iniciais
- `users`: cria usuario admin

Todas as seeds rodam apenas quando a tabela estiver vazia.
