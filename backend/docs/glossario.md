# Glossario (Leigo)

Este glossario explica termos tecnicos em linguagem simples.

- **API**: um conjunto de enderecos que o sistema usa para enviar e receber dados.
- **Endpoint**: um desses enderecos (ex: `/api/users/`).
- **Backend**: a parte que fica no servidor, onde os dados e as regras moram.
- **Frontend**: a parte que aparece na tela para o usuario.
- **Banco de dados**: local onde os dados ficam salvos.
- **Migration**: historico de mudancas do banco. Ajuda a criar/alterar tabelas automaticamente.
- **Seed**: insercao automatica de dados iniciais.
- **CRUD**: operacoes basicas: criar, listar, atualizar e deletar.
- **Token**: codigo usado para autenticar um usuario.
- **CORS**: regra que permite o frontend acessar o backend quando estao em enderecos diferentes.
- **MVC**: organizacao do codigo em Model (tabela), Controller (regras) e Routes (rotas).
- **DDD**: organizacao por modulos (dominios) para manter o projeto limpo.
